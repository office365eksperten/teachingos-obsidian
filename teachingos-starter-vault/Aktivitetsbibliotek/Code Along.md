# Code Along

## Formål

Bygge tryghed ved at alle producerer kørende kode sammen.

## Sådan gør du

1. Del skærm og skriv langsomt.
2. Vent på at alle er med efter hvert trin.
3. Indlæg små "prøv selv"-pauser.

## Varianter

Giv en delvist udfyldt startfil for at spare tid.
