# Live Coding

## Formål

Vise problemløsning i realtid — inkl. fejl og fejlfinding.

## Sådan gør du

1. Start fra en tom fil.
2. Tænk højt ved hvert valg.
3. Lav bevidst en typisk fejl og fejlfind den sammen.
4. Bed løbende de studerende gætte næste linje.

## Varianter

Lad de studerende kode med (pair-programming ved siden af).
